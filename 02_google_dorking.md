# Google Dorking

- Google Dorking means using advanced search operators to find hidden information
- Also called Google Hacking

## Common Operators
- site: → search within a specific website
- intitle: → search in page title
- inurl: → search in URL
- filetype: → search for specific file types
- intext: → search inside page content
- cache: → see cached version of a page
- link: → find pages linking to a URL

## Examples of Use Cases
- Finding exposed login pages
- Finding open directories
- Finding sensitive documents (PDF, XLS, DOC)
- Finding vulnerable servers
