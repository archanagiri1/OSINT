# OSINT Tools List

## Search and Discovery
- Maltego → link analysis and visualization
- Recon-ng → web reconnaissance framework
- SpiderFoot → automated OSINT collection
- theHarvester → emails, subdomains, IPs from public sources
- Shodan → internet connected device search engine
- Censys → internet wide scanning and search

## Username and Email
- Sherlock → username search across 300+ platforms
- Maigret → username search across 2500+ sites
- Holehe → check email registration on websites
- Hunter.io → find emails linked to a domain
- HaveIBeenPwned → check email in data breaches
- EmailRep.io → email reputation and linked accounts
- Epieos → find Google account linked to email

## Domain and DNS
- Whois → domain registration lookup
- Dig → DNS record query tool
- Nslookup → DNS lookup tool
- Sublist3r → subdomain enumeration
- Amass → advanced subdomain enumeration
- DNSEnum → automated DNS enumeration
- DNSRecon → advanced DNS reconnaissance
- DNSDumpster → DNS recon and research tool
- SecurityTrails → historical DNS and domain data
- crt.sh → certificate transparency log search

## IP Address
- Nmap → port scanner and network mapper
- Shodan → open ports and service detection
- Censys → internet wide IP scanning
- IPInfo.io → IP geolocation and ASN lookup
- AbuseIPDB → IP abuse and blacklist checker
- VirusTotal → IP threat intelligence
- BGP.he.net → ASN and IP range lookup
- GreyNoise → internet background noise analysis

## Image Investigation
- ExifTool → extract image metadata

- FotoForensics → image manipulation detection
- Forensically → advanced image forensics
